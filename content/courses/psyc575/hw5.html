Learning Objective

  • Evaluate published research that uses multilevel modeling

Assignment Description

In this assignment, you will evaluate a published article, presumably the article you read in HW 1 (but you can use another article), using the Desiderata table in McCoach (2019, pp. 293–294). Most likely you only need to read the method and the results section of the paper. Because we have not covered every topic in that table, you will be evaluating the article based on a subset of the desiderata. However, other points in the table are still important—just that we haven’t formally introduced them yet. The goal is that by analyzing an article, you will be able to identify what needs to be reported in your own multilevel analyses, for the class project and for your future research.

In addition, because we will need to do Bayesian MLM moving forward, it’s important you have the rstan package installed properly on your computer, as part of this assignment.

Steps

  1. Make a copy or Download the “HW 5 Template” file. One way to do it is to make a copy of the file to your own Google Drive.

Part 1

  1. Follow the instruction on https://github.com/stan-dev/rstan/wiki/RStan-Getting-Started to install the rstan package, which will be needed moving forward. Pay attention to the instruction as installing rstan is not as straight forward as installing other packages, depending on what operating system and what version of it you are using.

  2. After installing rstan, run the following and copy the output to the spreadsheet.

sessioninfo::package_info("rstan", dependencies = FALSE)
  1. Run the following in R, which, if run successfully, should give a graph. Copy the resulting graph to the spreadsheet to show that you have successfully installed the rstan package. Please make sure it works on your computer as future assignments will require using rstan.
library(rstan)
scode <- "
parameters {
  real y[2]; 
} 
model {
  y[1] ~ normal(0, 1);
  y[2] ~ double_exponential(0, 2);
} 
"
fit1 <- stan(model_code = scode, iter = 10, verbose = FALSE) 
plot(fit1)

Part 2

  1. Find an article that uses multilevel modeling in your field. It can be the article you used for HW 1.
  2. Imagine you’re a reviewer for the article. From the list of desiderata on the template, evaluate whether the article meets each of the reporting requirement.
    1. If the article meets the requirement, copy the sentences that are relevant. For tables/figures, you can cut the picture into the spread sheet.
    2. If the article does not report the required information, write a sentence to suggest what the authors can do. You can use the wordings from the desiderata.
  3. Double check to make sure you have one cell filled for each row (i.e., whether Column B or Column C).
  4. Submit your work to Blackboard. It can be in Excel or in PDF (but make sure all your answers are shown).
